# WC2026_Sticker_Tracker
Simple R Shiny to track your sticker collection

## How to use
Quite simple really, just download the starting dataset, then upload that, then add in stickers either by the players name, or by using the code on the back of the sticker. 

If you make a mistake you can remove stickers on the manage card tab. 

## How to save
Click the save changes button on the LHS and then when you come back re-upload that file instead of the starting one. 

## How can I visualise 
Sticker book has a virtual progress checker to look at.
